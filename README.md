Este projeto é um site para restaurante, a ideia foi criar um sistema simples, mas funcional, que permitisse que um cliente entrasse na plataforma, visse os pratos disponíveis e conseguisse fazer um pedido completo.

O usuário pode criar uma conta ou fazer login, e depois disso ele consegue navegar pelo menu, adicionar itens no carrinho, alterar quantidades e remover itens. Quando terminar, ele pode finalizar a compra.
Todas essas ações ficam registradas no banco de dados, então o sistema mantém as informações mesmo depois que o usuário sai do site.

O foco principal do projeto foi praticar HTML, CSS e JavaScript no frontend e ver como funciona a integração com um banco de dados real usando o Supabase.

Tecnologias Utilizadas

Frontend:
HTML5
CSS3
JavaScript

Backend / Banco de Dados:
Supabase (PostgreSQL, autenticação e API)
Operações de CRUD feitas diretamente pelo Supabase

Usamos essas tecnologias porque são acessíveis e permitem montar um sistema completo.

Instruções de Uso

Abra o arquivo index.html no navegador.
Na página inicial, faça login ou crie sua conta.
Depois de logado, navegue pelo cardápio para ver os pratos disponíveis.
Clique em adicionar ao carrinho para colocar o prato que deseja comprar.
No carrinho, é possível aumentar ou diminuir a quantidade ou remover algum item.
Após revisar tudo, basta finalizar a compra.
Todas as ações são enviadas e salvas no Supabase automaticamente.
